Glossary

Below is a glossary of terms that will be used in the quiz. Each one will link to the Mozilla Developer Network (MDN) so that you can learn more.

Variable

Variables are namespace containers for values. They can contain strings, numbers, functions, objects, arrays, and more.

Object Literal

In JavaScript, an object literal is a series of zero or more properties and key values that. This is comparable to a hashtable or dictionary in other languages.

Constructor

A constructor, or constructor function, is a function template for creating objects in JavaScript.

For... Loop

A for loop is a program flow construct that repeats its code block depending on the params that it is passed.

Array

An array is a series of zero or more elements contained in a list like format.

Helper Function

A helper function is a style of function that is intended to be reused by other functions for one concise purpose. It differs in no way from a regular function except in how it is implemented.

Switch Statement

A switch statement checks an expression against a series of case clauses and executes the inside of the case if it finds a match.

AJAX

AJAX is a collection of technologies that are used for asynchronous calls to and across servers.
